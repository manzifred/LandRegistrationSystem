# Land Registration and Ownership Tracking System


#Overview

This project is a Java-based Land Registration and Ownership Tracking System designed to digitize land records and track ownership changes securely.

#Features

Officer Login
Owner Registration
Land Registration
Ownership Transfer
Land Search
Ownership History Tracking
#Architecture

The system follows MVC architecture:

Model: Represents data entities
View: Swing GUI interfaces
Controller: Business logic handling
